## 2.2.安装SDK

我们需要把api_SDK中的两个依赖装入本地maven中，进入api_sdk目录，有两个项目需要处理：

![1532941356724](assets/1532941356724.png)

打开cmd命令行，输入命令：

```powershell
mvn install -Dmaven.test.skip=true -Dgpg.skip=true
```

然后进入另一个项目，上面的操作执行一遍
